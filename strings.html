String Manipulation in Python

Strings are a fundamental data type in Python, used to store text. This tutorial will cover basic string operations.

Creating Strings

Strings in Python can be created using single, double, or triple quotes. Here are examples of each:

# Single quotes
single_quote_string = 'Hello, World!'

# Double quotes
double_quote_string = "Hello, World!"

# Triple quotes (for multi-line strings)
triple_quote_string = '''This is a multi-line string.
It spans multiple lines.'''

Accessing Characters

You can access characters in a string using their index. Python uses zero-based indexing, so the first character is at index 0.

# Accessing characters in a string
print(single_quote_string[0])  # Output: H
print(double_quote_string[7])  # Output: W

Here are some common methods used for string manipulation:

  • concat(): Concatenates two strings.
  • slice: Extracts a portion of the string.
  • upper() and lower(): Converts the string to upper or lower case.
# Concatenating strings
concatenated_string = single_quote_string + " " + double_quote_string
print(concatenated_string)  # Output: Hello, World! Hello, World!

# Slicing a string
sliced_string = concatenated_string[7:12]
print(sliced_string)  # Output: World

# Upper and lower case conversion
print(single_quote_string.upper())  # Output: HELLO, WORLD!
print(double_quote_string.lower())  # Output: hello, world!
