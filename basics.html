Array Manipulation

Arrays are used to store multiple values in a single variable.

import numpy as np


Arrays     come in different types, such as single-dimensional arrays and multi-dimensional arrays. Common operations on arrays include:


    

  • Insertion: Adding an element to the array.
    • 

  • Deletion: Removing an element from the array.
    • 
  
  • Traversal: Accessing each element of the array.
    • 



Here is an example of a multi-dimensional array:


import numpy as np

# Creating a 2D array
array_2d = np.array([[1, 2, 3], [4, 5, 6]])

# Accessing an element
element = array_2d[0, 1]  # Accesses the element in the first row, second column
print(element)  # Output: 2


array = np.array([1, 2, 3, 4, 5])
array[0] = 10
print(array)
More on Array Manipulation

Arrays can be manipulated using various operations:

  • Sorting: Sorting elements in ascending or descending order.
  • Searching: Finding an element in the array.
  • Filtering: Selecting elements based on a condition.

# Sorting an array
sorted_array = np.sort(array)
print(sorted_array)  # Output: [1, 2, 3, 4, 5, 6]

# Searching for an element
index = np.where(array == 3)
print(index)  # Output: (array([0]),)

# Filtering elements greater than 3
filtered_array = array[array > 3]
print(filtered_array)  # Output: [4, 5, 6]

    Strings can be manipulated using various operations:
    

      

    • Replacing: Replacing a substring with another string.

# Replacing a substring
text = "Hello, World!"
replaced_text = text.replace("World", "Python")
print(replaced_text)  # Output: Hello, Python!

# Splitting a string
text = "apple,banana,orange"
fruits = text.split(",")
print(fruits)  # Output: ['apple', 'banana', 'orange']

# Joining a list of strings
fruits = ['apple', 'banana', 'orange']
joined_text = ", ".join(fruits)
print(joined_text)  # Output: apple, banana, orange
