Basic Coding Concepts

Variables

Variables are containers for storing data values.

let x = 5;
let y = 10;
let z = x + y;

Control Structures

Control structures determine the flow of execution in a program.

if (x > y) {
    console.log("x is greater than y");
} else {
    console.log("x is not greater than y");
}
