Array Manipulation in Python

Arrays are a fundamental data structure in Python, allowing you to store multiple items in a single variable. In Python, arrays can be implemented using lists or the array module. This tutorial will cover basic array operations.

Creating Arrays

In Python, you can create an array using the array module or by using lists. Here are examples of both:

import array

# Creating an array using the array module
arr = array.array('i', [1, 2, 3, 4, 5])
print(arr)

# Creating an array using a list
arr_list = [1, 2, 3, 4, 5]
print(arr_list)

Accessing Elements

You can access elements in an array using their index. Python uses zero-based indexing, so the first element is at index 0.

# Accessing elements in an array
print(arr[0])  # Output: 1
print(arr_list[2])  # Output: 3

Common Array Methods

Here are some common methods used for array manipulation:

  • append(): Adds an element to the end of the array.
  • remove(): Removes the first occurrence of a specified element.
  • slice: Extracts a portion of the array.
# Appending an element
arr.append(6)
print(arr)  # Output: array('i', [1, 2, 3, 4, 5, 6])

# Removing an element
arr.remove(3)
print(arr)  # Output: array('i', [1, 2, 4, 5, 6])

# Slicing an array
sliced_arr = arr[1:4]
print(sliced_arr)  # Output: array('i', [2, 4, 5])
