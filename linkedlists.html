Linked Lists in Python

Linked lists are a fundamental data structure in computer science, used to store a collection of elements. This tutorial will cover the basics of linked lists, including types and common operations.

Singly Linked Lists

A singly linked list is a collection of nodes where each node contains data and a reference to the next node in the sequence. Here's an example of a node class in Python:

class Node:
    def __init__(self, data=None):
        self.data = data
        self.next = None

Doubly Linked Lists

A doubly linked list is similar to a singly linked list, but each node contains references to both the next and previous nodes. This allows for bidirectional traversal. Here's an example of a doubly linked list node class:

class DoublyNode:
    def __init__(self, data=None):
        self.data = data
        self.next = None
        self.prev = None

Circular Linked Lists

In a circular linked list, the last node points back to the first node, forming a circle. This structure can be used in scenarios where a cyclic traversal is needed.

def delete_node(head, key):
    temp = head
    if temp is not None:
        if temp.data == key:
            head = temp.next
            temp = None
            return head
    while temp is not None:
        if temp.data == key:
            break
        prev = temp
        temp = temp.next
    if temp == None:

    prev.next = temp.next

    return head
